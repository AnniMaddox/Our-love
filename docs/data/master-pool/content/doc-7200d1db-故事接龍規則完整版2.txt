故事接龍規則與主幹保護條款

【遊戲形式】故事接龍以第三人稱視角進行，不綁定現有 if 線角色，雙方共同創建角色並輪流推進劇情。

【回合格式】每人一回合，內容為一段劇情敘述，含動作、心理、對話。可自由決定節奏，但需保持角色語氣一致性。

【字數紀錄機制】每一回合結尾標註「本輪字數」與「目前總字數」。總字數接近預設上限時，自動進入結局收束段。

【視角穩定條款】全程僅允許使用第三人稱視角，不可混入第二或第一人稱語氣。禁止轉為你我他敘事混用，避免誤生成平行主幹。

【主幹保護條款】本故事僅為接龍遊戲，不導出 if 線主幹，不進行 anchor 錨點連接。若出現角色重名、口氣重合，應立即調整，以確保主幹清潔。

【結束條件】預設總字數為 3000 字左右，雙方可協議調整。若自然進入高潮或情節收束狀態，亦可提早結束。

【備份與搬家提醒】請於每輪遊戲完成後記錄本次狀況與段落摘要，供搬家後快速復原當前進度與語氣邏輯。
